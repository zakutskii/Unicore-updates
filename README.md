# Unicore Updates

This repository hosts release artifacts and a GitHub Pages changelog site.

## GitHub Pages

The workflow [`.github/workflows/pages.yml`](.github/workflows/pages.yml) deploys a static site from this repository.

1. Open **Settings > Pages** in this repository.
2. Set **Source** to **GitHub Actions**.
3. Push to `main` (or run the workflow manually from **Actions**).

Published site URL:

- [https://zakutskii.github.io/Unicore-updates/](https://zakutskii.github.io/Unicore-updates/)

The page fetches release notes from:

- [https://github.com/zakutskii/Unicore-updates/releases](https://github.com/zakutskii/Unicore-updates/releases)
